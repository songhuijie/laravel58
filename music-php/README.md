##  Requirement

* PHP >= 5.6
* Composer

## Installation

### Global installation

``` shell
$ composer global require guanguans/music-php
```

### Current directory installation

``` shell
$ composer create-project guanguans/music-php
```

## Usage

``` shell
$ ./path/music-php
```
